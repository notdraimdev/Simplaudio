# Simplaudio
Simplaudio is a offline simple to use Music Player with support for downloading and discord rich presence.

![image](https://github.com/user-attachments/assets/071a962f-b450-4e20-99e1-ba01df3c555e)


## Features
- Downloading

Download any songs or playlists from youtube

![image](https://github.com/user-attachments/assets/f5831068-7c72-4ba3-81ce-ec6b30d777e5)

- Audio effects

Reverb, EQ or compression are accesible in the settings menu

- Playlists

Have multiple places to store your music with playlists
